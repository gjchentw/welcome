# Data Model: Rename Build Artifact to Welcome.jar

本功能不涉及資料庫實體，僅涉及檔案系統的命名規範。

## Artifact Naming Convention

| 產物類型 | 格式 | 範例 |
| :--- | :--- | :--- |
| 主程式 JAR | `Welcome-<version>.jar` | `Welcome-1.1.0.jar` |
| 原始碼包 | `Welcome-<version>-sources.jar` | `Welcome-1.1.0-sources.jar` |
| 文件包 | `Welcome-<version>-javadoc.jar` | `Welcome-1.1.0-javadoc.jar` |
